字符串。相比c语言里面的char[]可是好用多了。

2. string的4种初始化

  1. string s1;
  2. string s2(s1);
  3. string s3(“value”);
  4. string s4(n, ‘c’);

3. string的操作

操作 解释
s.empty() 是否为空串，返回true或者false if (st.emtyp()) ..
s.size() 字符串长度，相当于java里面的length()
s[n] 数组一样的用法
