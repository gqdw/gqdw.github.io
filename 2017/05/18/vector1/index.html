3. string的操作

操作 解释
s.empty() 是否为空串，返回true或者false if (st.emtyp()) ..
s.size() 字符串长度，相当于java里面的length()
s[n] 数组一样的用法
s1 + s2 连成一个新的字符串，返回
s1 = s2 s1的内容替换成s2的副本
v1 == v2 比较内容是否相等
> < != 比较每个字符，比较字符串会像c语言里面的调用strcmp？不存在的。
