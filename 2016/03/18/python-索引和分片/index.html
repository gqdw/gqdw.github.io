python 索引和分片

2016-03-18

index和c一样，从0开始，但是可以用负数，比如-1， -2 。意思是从尾部开始的偏移量。

S = ‘spam’
S[-1] -> ‘m’

分片

S[i:j]

意思提取是从i到j的部分作为一个序列返回。不包括S[j].

S[i:j:k] 还可以设置步进，
S[::-1] 就是逆序输出了。

L[:] 相当于做了一个拷贝，

还有一个分片的典型场景是处理命令参数
